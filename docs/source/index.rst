Welcome to the XAS Tools documentation!
=======================================

This package allows basic automation of X-ray absorption spectroscopy
simulations with the supercell core-hole approach [1].  The package also
provides functionality for the postprocessing of calculation results.

[1] F. Karsai, M. Humer, E. Flage-Larsen, P. Blaha and G. Kresse,
    *Phys. Rev. B* **98**, 235205 (2018).
